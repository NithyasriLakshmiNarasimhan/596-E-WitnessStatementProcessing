To Select Python Environment: 

in Visual Studio press ctrl + shift + p
choose: Python: Select Interpreter 
select the .venv in this project location.

To run the Flask side:
Restart the terminal after having done the previous step.
Navigate to \596-E-WitnessStatementProcessing\QandA
run the command 'flask run'

To run the react app:
Open a new terminal, seperate from the flask terminal
Navigate to \596-E-WitnessStatementProcessing\FrontEnd\reactapp
run the command 'npm install --legacy-peer-deps' to install the react app
run the command 'npm start'
